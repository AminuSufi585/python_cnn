Introduction

Convolutional networks were inspired by biological processes in which the connectivity pattern between neurons is inspired by the organization of the animal visual cortex. Individual cortical neurons respond to stimuli only in a restricted region of the visual field known as the receptive field. The receptive fields of different neurons partially overlap such that they cover the entire visual field. I built network this using only Numpy and it has has multiple layers which makes it a deep learning. I pluged the model to a Django App and used JavaScript to capture the input so it can be used as a web app. I pre-trained the model to save time.

What the heck is CNNs for ?

Convolutional neural nets are a weird combination of biology and math together with CS, and these networks have been some of the most exciting innovations in the field of computer vision. Lots of big companies have been using deep learning(AKA deep neural net) at the core of their services.
Facebook uses neural nets for their automatic tagging algorithms, Google for their photo search, Amazon for their product recommendations, Pinterest for their home feed personalization, and Instagram for their search infrastructure. These networks also enable vision in robotics and driverless cars.
To understand convolutional neural networks (CNNs), you must first understand convolution. Wikipedia gives a good mathematical definition here.

Why I did this

There a many machine learning libraries that simplify the maths and most of the challenges people fear about AI. Such packages as TensorFlow, Theano, Pytorch. But I always wanted to know how the abtractions these package have built in occur. I choose to learn the math and logic using just numpy, which is a very convenient package for matrix manipulation. I started with the Keras MNIST convolutional neural net example and then made modifications to create this project.
